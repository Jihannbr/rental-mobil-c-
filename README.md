# 🚗 Car Rental Console App (C++)

## 🇮🇩 Deskripsi (Indonesia)
Program ini merupakan aplikasi rental mobil sederhana berbasis console yang dibuat menggunakan bahasa C++.  
Pengguna dapat memilih jenis kendaraan, menentukan lama sewa, dan sistem akan menghitung total biaya, diskon, serta kembalian pembayaran.

### ✨ Fitur
- Memilih jenis kendaraan
- Menghitung total biaya sewa
- Penerapan diskon berdasarkan lama sewa
- Input pembayaran dan menghitung kembalian

---

## 🇬🇧 Description (English)
This is a simple console-based car rental application built using C++.  
Users can select a vehicle, input rental duration, and the system will calculate total cost, discount, and payment change.

### ✨ Features
- Select vehicle type
- Calculate total rental cost
- Apply discount based on rental duration
- Payment input and change calculation
